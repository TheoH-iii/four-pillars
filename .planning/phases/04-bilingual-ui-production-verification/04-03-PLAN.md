**buildCardFiveElements() element names in sorted array:**
The sorted array uses ELEMENT_NAMES[el] which returns English names. In zh mode these should stay as Chinese element names. Add a helper:
```js
function elLabel(el) {
  if (window.currentLanguage === 'zh') {
    var ZH_EL = { wood:'木', fire:'火', earth:'土', metal:'金', water:'水' };
    return ZH_EL[el] || el;
  }
  return ELEMENT_NAMES[el] || el;
}
```
Replace ELEMENT_NAMES[el] calls inside buildCardFiveElements with elLabel(el).
Also replace formatElementNames() calls in buildCardLuckyElements and buildCardLifeGuidance with a version that uses elLabel:
```js
function formatElementLabels(arr) {
  return (arr || []).map(function(el) { return elLabel(el); });
}
```
Use formatElementLabels() in place of formatElementNames() for display in those two cards.

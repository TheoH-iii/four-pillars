  <name>Task 1: Implement streaming Claude proxy in api/reading.js</name>
  <files>api/reading.js</files>
  <action>
Replace the stub with a real implementation. The function must:

1. Reject non-POST requests with 405.
2. Parse JSON body: `{ chartSummary, cardType, language }`.
   - `chartSummary`: compact object (pillars, dayMaster, strength, pattern, favorableElements, cardData)
   - `cardType`: string identifying which card (e.g. "overview", "dayMaster", "fiveElements", etc.)
   - `language`: "en" or "zh" (default "en" if absent)
3. Set SSE headers before writing anything:
   ```
   Content-Type: text/event-stream
   Cache-Control: no-cache
   Connection: keep-alive
   X-Accel-Buffering: no
   ```
4. Build the Claude API request:
   - Model: `claude-3-5-haiku-20241022` (fast, cost-effective for streaming)
   - max_tokens: 600
   - stream: true
   - system prompt: establish persona as a master Bazi astrologer — a human fortune teller giving a live interpretation. Never mention AI, Claude, or "based on your data". Tone: wise, direct, personal. If language is "zh", instruct to respond in Traditional Chinese.
   - user message: include the compact chartSummary as JSON, then ask for a reading specific to cardType. For "overview" card, include narrative intro + career/relationships/health/wealth sections. For all other cards, one focused paragraph on that aspect.
5. Call `https://api.anthropic.com/v1/messages` with:
   - Header `x-api-key: process.env.ANTHROPIC_API_KEY`
   - Header `anthropic-version: 2023-06-01`
   - Header `content-type: application/json`
   - Body: the request object above with `stream: true`
6. Read the response body as a stream. For each line:
   - Skip empty lines and lines not starting with `data: `
   - Parse the JSON after `data: `
   - If `type === "content_block_delta"` and `delta.type === "text_delta"`, write: `res.write('data: ' + JSON.stringify({ token: delta.text }) + '\n\n')`
   - If `type === "message_stop"`, write: `res.write('data: [DONE]\n\n')` then `res.end()`
7. On any error (network, API key missing, non-200 from Anthropic), end the SSE stream with an error event: `res.write('data: ' + JSON.stringify({ error: 'Reading unavailable' }) + '\n\n')` then `res.end()`.

Use Node.js built-in `https` module or native `fetch` (available in Node 18+ on Vercel). Do NOT use the Anthropic SDK — keep dependencies at zero. Use `fetch` since Vercel Node 18 runtime supports it natively.

The function signature must remain `module.exports = async (req, res) => { ... }`.
